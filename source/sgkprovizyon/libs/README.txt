================================
Google AdMob Ads SDK for Android
================================

This is the Google AdMob Ad SDK for Android.

The SDK provides:
- AdView, suitable for showing banner ads in many different sizes.
- InterstitialAd which provides a means to display interstitial ads (image and
video).

Requirements:
- An AdMob site ID or DoubleClick for Publishers account.
- Runtime of Android 1.5 or later.

Additional information is available in the documentation located in the docs
directory in this zip file.

You can also find the latest, detailed information about the SDK as well as
download examples using the SDK at:
https://developers.google.com/mobile-ads-sdk/docs/
